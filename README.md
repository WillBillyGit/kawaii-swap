# Kawaii Swap 🌸✨

**Multi-Chain Transmutation Bridge & Aether Vault**

Kawaii Swap is a mystical, high-performance decentralized application (dApp) designed for seamless asset transmutation across multiple blockchain realms. It combines a playful, cosmic aesthetic with production-grade Web3 infrastructure.

## 🌌 Features

- **Transmutation Hub**: Swap assets across Polygon, Base, BSC, Optimism, Avalanche, and Arbitrum.
- **Aether Vault**: Manifest and dissolve essence directly in the vault to participate in the protocol's liquidity flow.
- **Dynamic Pricing**: Custom-built Alchemy RPC fallbacks and Thirdweb integration for real-time rates and secure transactions.
- **Multimodal UI**: Responsive design with state-driven visual themes (Transmute / Vault / Tokens).

## 🛡️ Security & Transparency

Due to the use of the `crustfundfi.org` domain (originally registered for a bakery-themed project), we maintain full transparency through:
- `humans.txt`: Detailed team and project colophon.
- `security.txt`: Vulnerability reporting guidelines.
- `robots.txt`: Proper crawler instructions and project clarification.

## 🛠️ Technology Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS, Motion (framer-motion)
- **Web3**: Thirdweb SDK, Wagmi, Viem
- **Infrastructure**: Alchemy RPC nodes

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/WillBillyGit/kawaii-swap.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**:
   Create a `.env` file based on `.env.example` with your `VITE_THIRDWEB_CLIENT_ID` and `VITE_ALCHEMY_API_KEY`.
4. **Run development server**:
   ```bash
   npm run dev
   ```
